## Prerequisites

- **Python**: 3.11
- **Hardware**: Multi-GPU setup (tested on GB200, H100)
  - Minimum: 2 GPUs for distributed inference
- **CUDA**: Compatible GPU with CUDA 12.9+

## Run


```bash
# tip: use uv to setup env following the README
# tip: reserve larger memory

sh ~/salloc_h100.sh --mem 840G --time 00:30:00 --gres gpu:a100:2 #注意是80G的a100
# then
export HF_HOME="/home/zz9706/.cache/huggingface"          # where hf downloads to / reads from
export TRANSFORMERS_OFFLINE=1                    # never hit the internet
export HF_DATASETS_OFFLINE=1           
export HF_HUB_OFFLINE=1  # belt-and-suspenders    
TORCHDYNAMO_VERBOSE=1 CUDA_VISIBLE_DEVICES=0,1 uv run python -m torch.distributed.run --standalone --nproc_per_node=2 socket_test_optimized_AR.py --port 5000 --enable-dit-cache --model-path  /home/zz9706/.cache/all_checkpoints/dreamzero

# in another shell window, 
uv run python test_client_AR.py --port 5000
```
