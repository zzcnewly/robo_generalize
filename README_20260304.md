## Evaluation Protocol

### Big picture
1. Start the Model to listen at port e.g., 6000

2. Client call port e.g., 6000. (via Socket; send input images and prompts; then receive action results)

## Transfer training data to GEAR format
source: libero_plus training data
```bash
uv run python scripts/data/convert_lerobot_to_gear.py \
    --dataset-path .cache/data/libero_plus_lerobot/libero_plus_lerobot \
    --output-path .cache/data/libero_plus_lerobot/libero_plus_lerobot_converted \
    --embodiment-tag libero_sim \
    --state-keys '{"eef_position": [0, 6], "gripper_position": [6, 8]}' \
    --action-keys '{"eef_position": [0, 6], "gripper_position": [6, 7]}' \
    --relative-action-keys eef_position \
    --task-key annotation.task \
    --action-horizon 8
```
